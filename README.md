# Simulate CI CD Pipeline Flow

This Python script simulates a simplified CI/CD (Continuous Integration/Continuous Delivery) pipeline. It demonstrates the automated steps involved in fetching code, building the project, running tests, and deploying to staging and production environments. The script highlights how automation ensures code quality and faster, more reliable deployments by stopping the pipeline at any failed stage.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Open a terminal or command prompt.
3. Navigate to the directory where you saved `main.py`.
4. Run the script using `python main.py`.
5. Optionally, modify the `SIMULATE_*_FAILURE` flags at the top of `main.py` to observe different pipeline failure scenarios.
